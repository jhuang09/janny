This is something cool you can make with CSS

CSS is what makes pages look mad ~ * a e s t h e t i c * ~

But it can also help create a template for your entire website! It makes sure all the pages in your website look real professional and user friendly. :)

From colors to the navigation bar, everything you're seeing here is all controlled by CSS and it makes my HTML code look much nicer and easier for someone else to read.

I also learned HTML and CSS in the span of 1.5 days, so please allow me to feel like I've accomplished something with this trashy website.

Why CSS is important

  • It freaking controls how the webpage looks! Isn't that awesome?!
  • You can change one thing in CSS and it'll automatically apply to everything else!
  • It can totally change how you view a website. Making good use of CSS and designing how the site looks can really affect what kind of impression you leave on the user.
